# Impeccable Shoes Tacloban Branch Website

This is a complete redesign of the Impeccable Shoes Tacloban Branch website, inspired by Sneaker Buff's design patterns. The site is a single-page, production-ready website for a premium shoe care and cleaning service franchise.

## Features

- **Single-Page Scroll Navigation:** with a fixed header for easy navigation.
- **Three-Pillar Service Structure:** Clean | Restore | Protect.
- **Trust Statistics Section:** showcasing the number of customers, pairs restored, and years of experience.
- **Before/After Image Gallery:** with a click-to-toggle feature to compare the results.
- **Multiple CTA Options:** Book, Call, Message, and Visit.
- **Testimonials Section:** featuring reviews from local customers.
- **Work Preview Grid:** showcasing the quality of the work.
- **Mobile Responsive:** with a hamburger menu for mobile navigation.
- **Smooth Animations:** and transitions for a premium feel.

## Technologies Used

- HTML5
- CSS3 (vanilla)
- JavaScript (vanilla)

## How to Use

1. Clone the repository: `git clone https://github.com/your-username/impeccable-shoes-website.git`
2. Open `index.html` in your browser.
